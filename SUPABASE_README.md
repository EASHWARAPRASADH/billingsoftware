## 🚀 Quick Start (3 Steps)

### Step 1: Create Supabase Account (5 min)
```
1. Go to https://supabase.com
2. Sign up (free)
3. Create project: "billmanagement"
4. Save credentials
```

### Step 2: Setup Database (2 min)
```
1. Open Supabase SQL Editor
2. Copy SUPABASE_SETUP_SCRIPT.sql
3. Paste and run
4. Done! ✅
```

### Step 3: Install & Configure (10 min)
```bash
# Backend
cd backend
npm install @supabase/supabase-js

# Frontend
cd frontend
npm install @supabase/supabase-js

# Add credentials to .env files (see ENV_SUPABASE_TEMPLATE.md)
```
